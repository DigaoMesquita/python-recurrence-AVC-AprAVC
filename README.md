AprAVC is a Machine Learning framework designed to support stroke recurrence prediction using real-world clinical data.
The project organizes a unified experimental protocol that integrates: preprocessing, class imbalance handling, training of multiple families of supervised models, and comparative evaluation.

> Objective: to provide a reproducible pipeline for comparing balancing strategies and models using metrics appropriate to the clinical scenario.

---

## Principais funcionalidades

Key Features
- Preprocessing pipeline (data cleaning, encoding, normalization/standardization, etc.)
- Class imbalance strategies:
  - Oversampling
  - Undersampling
  - Hybrid techniques
- Training and comparison of supervised models (e.g., ensemble and classical methods)
- Evaluation using robust metrics:
  - ROC-AUC, Precision, Recall, F1-score
  - Confusion Matrix
